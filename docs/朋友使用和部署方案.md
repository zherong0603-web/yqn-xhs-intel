# 朋友使用和部署方案

更新时间：2026-06-15

## 最推荐方案

把这个工具部署成一个网页。

朋友不用装代码，不用打开终端，只要打开一个网址：

1. 打开网址。
2. 填自己的 TikHub API Key。
3. 点“测试连接”。
4. 填关键词。
5. 点“先估算费用”。
6. 点“确认运行”。
7. 下载 Excel。

## 为什么不建议直接发文件夹给朋友

发文件夹虽然看起来简单，但朋友电脑可能没有 Python，可能打不开 `.command`，也可能端口冲突。

对不懂代码的人来说，最稳定的方式就是一个网页链接。

## API Key 怎么保护

现在已经改成“朋友模式”：

- API Key 只保存在朋友自己的浏览器里。
- 不写进服务器代码。
- 不写进 GitHub。
- 朋友 A 和朋友 B 不会互相覆盖 API Key。

## 最快部署路线

推荐：

- GitHub 放代码。
- Render 部署网页。
- 朋友打开 Render 给的网址。

Render 官方支持从 GitHub 仓库里的 Dockerfile 自动构建部署。

## 你需要做什么

### 第 1 步：新建 GitHub 仓库

在你的 GitHub 账号里新建一个仓库，例如：

`YQN-XHS-Intel`

然后把仓库链接发给 Codex。

仓库链接长这样：

`https://github.com/zherong0603-web/YQN-XHS-Intel.git`

### 第 2 步：让 Codex 推送代码

Codex 会把当前版本推到 GitHub。

### 第 3 步：打开 Render

去：

`https://render.com/`

登录后选择：

`New` -> `Web Service`

### 第 4 步：连接 GitHub 仓库

选择刚才的仓库：

`YQN-XHS-Intel`

### 第 5 步：选择 Docker

部署类型选择：

`Docker`

项目里已经准备好了：

- `Dockerfile`
- `render.yaml`

### 第 6 步：部署

点击部署后，Render 会生成一个网址。

你把这个网址发给朋友就行。

## 朋友怎么用

朋友打开网址后：

1. 点右上角 `API Key`。
2. 粘贴自己的 TikHub API Key。
3. 点 `测试连接`。
4. 成功后点 `保存到当前浏览器`。
5. 关键词填 `墨西哥海外仓`。
6. 想要结果数量填 `3`。
7. 安全上限最多调用填 `30`。
8. 安全上限最多花费填 `0.05`。
9. 点 `市场素材搜索`。
10. 点 `先估算费用`。
11. 点 `确认运行`。
12. 下载 Excel。

## 重要提醒

- 朋友真实运行时，用的是朋友自己的 TikHub API Key 和余额。
- “先估算费用”不会抓数据，不会产生业务数据费用。
- “确认运行”才会真实调用 TikHub。
- 服务器上会临时生成 Excel 下载文件；不要在公共链接里处理特别敏感的内部资料。

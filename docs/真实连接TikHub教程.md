# 真实连接 TikHub 教程

更新时间：2026-06-15

这份教程只讲怎么点，不讲技术原理。

## 你要准备什么

只需要一样东西：

- TikHub API Key

你可以把它理解成“门钥匙”。没有这把钥匙，YQN 助手只能跑演示数据；有了钥匙，才能真实去 TikHub 调小红书接口。

## 第 1 步：打开 YQN 助手

双击：

`启动YQN小红书市场情报助手.command`

如果页面没有自动打开，就在浏览器打开：

`http://127.0.0.1:8765`

## 第 2 步：填写 API Key

在页面右上角点：

`API Key`

把 TikHub API Key 粘贴到：

`TikHub API Key`

然后点：

`保存到本机`

注意：

- API Key 只保存在你的电脑。
- 不会写进代码。
- 不会上传 GitHub。

## 第 3 步：测试连接

还是在 API Key 弹窗里，点：

`测试连接`

如果看到“连接成功”，说明钥匙能用。

如果失败，通常是：

- API Key 少复制了一段。
- API Key 前后多了空格。
- TikHub 账号余额或权限有问题。

## 第 4 步：做第一次真实测试

推荐这样填：

- 关键词：墨西哥海外仓
- 获取数量：3
- 测试模式：打开
- 最大调用次数：30
- 最高费用：0.05

然后点：

`市场素材搜索`

再点：

`先估算费用`

看清楚预计费用后，最后点：

`确认运行`

## 第 5 步：打开 Excel

跑完后，右侧会出现：

`点击下载 / 打开 Excel`

也可以去这个文件夹找：

`YQN_XHS_Intel/outputs/`

## 本次真实测试会调用什么

如果选择“市场素材搜索”，数量填 3，程序通常会调用：

- 搜索笔记：1 次
- 笔记详情：最多 3 次

当前查到的接口单价是：

- $0.01/次

所以预计最多：

- 4 次
- $0.04

如果你把最高费用设成 $0.05，就算哪里异常，超过保护线也会停止。

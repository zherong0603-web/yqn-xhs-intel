## 朋友只看演示，怎么用

1. 打开上面的链接。
2. 不用填写 API Key。
3. 关键词保持“墨西哥海外仓”，或者自己换一个词。
4. 点“市场素材搜索”。
5. 点“先估算费用”。
6. 点“确认运行”。
7. 页面会生成一个演示 Excel。

不填写 API Key 时：

- 不会真实调用 TikHub。
- 不会扣费。
- Excel 是演示数据，用来看产品效果。

## 朋友要真实测试，怎么用

1. 点右上角“API Key”。
2. 粘贴自己的 TikHub API Key。
3. 点“保存到当前浏览器”。
4. 点“测试连接”。
5. 填关键词和想要结果数量。
6. 点“先估算费用”。
7. 看清楚预计调用次数和预计费用。
8. 点“确认运行”。

真实测试时：

- API Key 只保存在朋友自己的浏览器里。
- 服务器文件不会保存 API Key。
- 达到“最多调用”或“最多花费”会自动停止。
- 程序不会自动充值。

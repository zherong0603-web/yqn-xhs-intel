# YQN 小红书市场情报助手 第一阶段执行计划

更新时间：2026-06-15

## 目标

做一个老板能看懂的大致 demo：
- 打开就是中文页面。
- 可以填 API Key、关键词、数量和费用上限。
- 先估算调用次数和费用。
- 你点“确认运行”后才开始。
- 跑完生成 Excel。
- 第一轮只用“墨西哥海外仓”，最多 3 篇笔记。

1. 写接口说明文档。

3. 做 API Key 中文设置页。
4. 做任务按钮和自动选接口规则。
5. 做运行前估费弹窗。
6. 做费用保护和调用次数保护。
7. 做缓存、去重、失败重试、中断续跑。

## 第一版技术选择

用本机小网页方式：
- 后端：Python Flask
- 前端：普通 HTML/CSS/JS
- 本地保存：JSON 文件和 SQLite
- Excel：openpyxl 生成 `.xlsx`

这样做的好处：
- 你不用学命令行。
- API Key 留在本机。
- 后续可以继续加按钮、加字段、加自动分析。

## 自动选接口规则

市场素材搜索：
- 先调用搜索笔记。
- 只对目标数量内的笔记查详情。
- 详情接口按笔记类型选择图文或视频。

竞品账号研究：
- 先搜索用户。
- 再查用户信息。
- 再查用户发布笔记。

评论需求分析：
- 先搜索笔记。
- 再查前几篇笔记评论。
- 发现重点评论后，才查二级评论。

话题趋势研究：
- 先搜索笔记找话题线索。
- 有话题 ID 再查话题详情和话题笔记。

商品与用户痛点研究：
- 先搜索商品。
- 再查少量商品详情、评论总览、评论列表。

图片补充搜索：
- 只调用搜索图片。

群聊搜索：
- 只调用搜索群聊。

## 第一轮测试

默认测试参数：
- 关键词：墨西哥海外仓
- 数量：3
- 测试模式：开启
- 最大调用次数：30

- 花费显示为 0 美元。

如果填了 API Key：

- 你确认后才调用 TikHub。
- 最多拿 3 篇笔记。
- 预计最多调用 4 次，按当前单价约 $0.04。

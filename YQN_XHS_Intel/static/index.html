真实连接

照着 4 步走，就能真实测试

  1. 1
    去 TikHub 用户中心复制 API Key

    它像门钥匙。没有钥匙时，本工具只会跑演示数据，不会扣费。

  2. 2
    点右上角 API Key，粘贴并保存到本机

    保存位置在本机 `data/config.json`，不会写进代码，也不会上传 GitHub。

  3. 3
    先点“测试连接”，再点“先估算费用”

    连接成功后，工具会实时查接口单价，算出这次最多花多少钱。

  4. 4
    最后点“确认运行”

    第一轮建议：关键词“墨西哥海外仓”，数量 3，最高费用 $0.05。
